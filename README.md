# Smart Expense AI

## Track Expenses • Predict Spending • Save Smarter

Smart Expense AI is a simple yet intelligent expense management web application that helps users track their spending, understand financial habits, and make smarter budgeting decisions.

In today's fast-paced lifestyle, people often lose track of where their money is going. Small daily expenses such as food orders, shopping purchases, subscriptions, and travel costs gradually add up, making it difficult to stay within budget. While many expense-tracking applications record past transactions, they rarely help users understand future spending patterns.

Smart Expense AI was developed to address this problem. The application not only tracks expenses but also analyzes spending behavior, predicts future expenses using rule-based AI logic, and provides personalized saving recommendations. By combining financial insights with visual analytics, the platform helps users stay aware of their spending habits and avoid unnecessary expenses.



## Overview

Smart Expense AI is designed to provide users with a smarter approach to personal finance management. The system combines expense tracking, expense prediction, visual analytics, overspending alerts, and saving recommendations within a single user-friendly platform.

The goal is to help users monitor their finances more effectively, understand where their money is being spent, and take proactive steps toward better financial planning.



## Key Features

### Expense Tracking

* Add and manage daily expenses easily.
* Organize expenses into categories such as:

  * Food
  * Shopping
  * Travel
  * Subscriptions
  * Others

### Expense Prediction

* Predicts next month's expenses based on previous spending patterns.
* Helps users plan budgets more effectively.

### Overspending Alerts

* Detects when spending exceeds ₹5000 in selected categories.
* Displays visual warnings and notifications to alert users.

### Smart Saving Tips

* Provides personalized recommendations based on spending behavior.
* Encourages better financial habits through practical suggestions.

### Data Visualization

* Category-wise Pie Charts
* Monthly Expense Bar Charts
* Color-coded spending indicators
* Interactive visual representation of financial trends

### Local Data Storage

* Uses browser LocalStorage for storing transactions.
* No internet connection or login required.
* Fast, lightweight, and easy to use.



## How It Works

1. The user enters expense details into the application.
2. Expense data is stored locally using LocalStorage.
3. The system analyzes spending patterns and category-wise expenses.
4. Rule-based AI logic predicts future spending.
5. Alerts are generated when expenses exceed predefined thresholds.
6. Charts and graphs update automatically in real time.
7. Personalized saving tips are displayed based on spending behavior.
